Enable AI-powered reconnaissance by analyzing scoped HTTP traffic. This extension helps security professionals extract meaningful insights from requests and responses without manual inspection. By leveraging Burp AI, the extension can identify authentication mechanisms, summarize API behavior, and infer application workflows - transforming raw traffic into actionable intelligence. Prompt templates and scoped host selection make the process efficient and context-aware.

Features

  • AI-driven analysis of HTTP traffic within scope
  • Summarizes application structure, workflows, and potential risks
  • Supports custom prompts and predefined templates
  • Automatically detects and lists scoped hosts
  • Scope-aware: analysis is restricted to defined targets

Usage

  1. Open the AI Recon Assistant tab.
  2. Select a host from the list of scoped domains (click "Refresh" if needed).
  3. Enter a prompt or select a predefined question.
  4. Click "Ask AI" to initiate the analysis.
  5. Review the structured insights returned by the AI engine.
